# Customize

Overview of extension points for django-iubenda.

1. [Configuration (`APP_CONFIG`)](configuration.md) — central `APP_CONFIG["iubenda"]` and `IUBENDA_*` precedence.
2. [Templates](templates.md) — override privacy, cookie, and embed templates.
3. [HTTP client & policy API](http-api.md) — API base URL, language mapping, timeouts, and the shared HTTP client.
4. [Google Tag Manager](google-gtm.md) — `GTM` / `IUBENDA_GTM` and official Iubenda notes.

Further option lists (banner, consent flags, etc.) remain in the main [Optional settings](../index.md#optional) section on the home page.
